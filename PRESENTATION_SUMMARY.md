## üìã 6-Step User Journey

### 1. **Sign Up & Login**
- Create account with financial profile
- Access personalized dashboard

### 2. **Risk Assessment** ‚≠ê
- 6-question interactive questionnaire
- Algorithm calculates risk tolerance (1-10)
- Saves to profile automatically

### 3. **Browse Investments** ‚≠ê‚≠ê‚≠ê
- **Search & Filter**: Find stocks by ticker, name, sector, category
- **Real-Time Prices**: Live updates every 30 seconds with change indicators
- **Risk Indicators**: Visual bars showing volatility from price history
- **Explanations**: AI-generated "why this recommendation" for each stock
- **Charts**: Expected returns and risk analysis visualizations

### 4. **Plan Fund Allocation** ‚≠ê
- Divides funds into 4 buckets:
  - Emergency Fund (2% growth)
  - Short-term (4% growth)
  - Medium-term (6-8% growth)
  - Long-term (8-12% growth)
- Interactive controls: savings, contributions, time horizon, risk
- Stacked area chart showing growth over time
- Year-by-year breakdown table

### 5. **Compare Scenarios** ‚≠ê
- Three strategies: Conservative, Moderate, Aggressive
- Side-by-side line chart comparison
- Individual scenario cards with metrics
- Comparison table

### 6. **Validate Portfolio** ‚≠ê
- Pie chart visualization
- Real-time validation (errors/warnings/success)
- Portfolio statistics
- Asset details list
